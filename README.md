# Terminal Todo App

**Что должна уметь программа**
- Создавать туду
- Удалять туду
- Зачеркивать/расчеркивать туду
- Показывать список

**Требования**
- Все будет работать на основе команд
    - Удаление (удалить №туду)
    - Добавить
    - Зачеркнуть/расчеркнуть
    - Хелп

**Экраны**
- Список туду
    - В списке отображается весь список туду отсортированных по дате создания
    - Командная строка
