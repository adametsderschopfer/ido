# Terminal Todo App

## Что, почему и зачем?
- Работа с json
  - **serde**
  - **serde_json**
- **dialoguer** - Помогает делать интерактивный выбор внутри терминала
- **colored** - Добавляет к строкам методы покраски текста выводимый с помощью println!
- **clap** - Пакет помогающий облегчить работу с реализацией cli приложений


## Описание
**Что должна уметь программа**
- Создавать туду
- Удалять туду
- Зачеркивать/расчеркивать туду
- Показывать список

**Требования**
- Все будет работать на основе команд
    - Удаление (удалить №туду)
    - Добавить
    - Зачеркнуть/расчеркнуть
    - Хелп

**Экраны**
- Список туду
    - В списке отображается весь список туду отсортированных по дате создания
    - Командная строка
